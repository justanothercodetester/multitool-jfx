# Info
This program was built in using java, javafx, atlantafx, and fxml. Although my previous javafx projects used my own custom installer, I decided that my custom installer was too unstable and sometimes didn't work on some versions of windows. This should fix compatibility issues because it uses an msi file and a custom jre created using jlinker and compiled using jpackage
